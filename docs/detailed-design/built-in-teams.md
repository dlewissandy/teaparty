# Built-in Teams

Default workgroup definitions for the management catalog. Available to any project;
not added to a project's active team unless explicitly opted in.

## Teams

| Team | Lead | Description |
|------|------|-------------|
| [research](teams/research.md) | research-lead | Web, academic literature, patents, and video |
| [writing](teams/writing.md) | writing-lead | Content production across formats and registers |
| [editorial](teams/editorial.md) | editorial-lead | Prose quality, accuracy, style, and voice |
| [quality-control](teams/quality-control.md) | quality-control-lead | Functional correctness, test coverage, AI detection |
| [art](teams/art.md) | art-lead | Diagrams, vector graphics, and generative imagery |
| [analytics](teams/analytics.md) | analytics-lead | Data analysis and visualization |
| [planning](teams/planning.md) | planning-lead | Strategy, risk, and roadmap design |
| [intake](teams/intake.md) | intake-lead | Requirements gathering and intent alignment |
| [coding](teams/coding.md) | coding-lead | Implementation, testing, and code review *(existing)* |
| [configuration](teams/configuration.md) | configuration-lead | Agent, workgroup, and skills configuration *(existing)* |
