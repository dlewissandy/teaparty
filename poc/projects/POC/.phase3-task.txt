## Prerequisites

Phase 1 must be complete first. memory entries now have YAML frontmatter with fields:
  id, type, importance, phase, status, reinforcement_count, created, last_accessed

## Problem

memory_indexer.py has two issues:

1. CHARACTER-BASED CHUNKING: chunk_text() splits memory files into 1600-character overlapping
   windows regardless of entry boundaries. A single logical entry (frontmatter + body) may be
   split across multiple chunks, or one chunk may mix two different entries. This corrupts
   semantic coherence and prevents per-entry prominence scoring.

2. IMPLICIT EVERGREEN EXEMPTION: apply_temporal_decay() only decays entries whose source file
   has a date in its PATH (like .../memory/2026-01-15.md). Files named MEMORY.md, OBSERVATIONS.md,
   or ESCALATION.md have no date in the path, so infer_date_from_path() returns None and decay
   is SKIPPED. This means project-level memories never decay and accumulate without bound.
   The concrete failure: specification-phase memories in MEMORY.md were permanently evergreen
   and saturated retrieval context during implementation sessions.

def compute_prominence(
    importance: int,
    created: date_type | None,
    reinforcement_count: int,
    today: date_type | None = None,
    half_life_days: float = 30.0,
) -> float:
    """Compute entry prominence: importance * recency_decay * reinforcement_multiplier.

    - importance: 1-10 (default 5 for legacy entries)
    - recency_decay: exp(-ln(2)/half_life_days * age_days). If created is None, age=0 (no decay).
    - reinforcement_multiplier: 1 + log(1 + reinforcement_count)
    - All sources decay. There is no evergreen exemption.
    """
    if today is None:
        today = date_type.today()
    if created is not None:
        age_days = max(0, (today - created).days)
    else:
        age_days = 0  # treat as fresh if no date
    lambda_d = math.log(2) / half_life_days
    recency_decay = math.exp(-lambda_d * age_days)
    reinforcement_mult = 1.0 + math.log(1.0 + max(0, reinforcement_count))
    return importance * recency_decay * reinforcement_mult
