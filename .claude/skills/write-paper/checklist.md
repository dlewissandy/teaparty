# Pre-Submission Quality Checklist

Run through this checklist before declaring a manuscript ready for submission. Not every item applies to every paper — use judgment.

---

## Argument & Coherence

- [ ] The paper answers one clear research question or advances one clear thesis
- [ ] The contribution is stated explicitly in the Introduction
- [ ] Every section serves the central argument (no orphan sections)
- [ ] The Discussion interprets results, not just restates them
- [ ] Limitations are acknowledged honestly, not buried or minimized
- [ ] The Conclusion follows from the evidence presented (no overclaiming)

## Citations & References

- [ ] Every factual claim is backed by a citation or original evidence
- [ ] No orphan references (cited in text but missing from bibliography, or in bibliography but never cited)
- [ ] Citation format matches the target journal's style
- [ ] Key seminal works in the field are cited where relevant
- [ ] Recent work (last 3-5 years) is represented, not just classics
- [ ] Self-citation is proportionate and justified

## Figures & Tables

- [ ] Every figure and table is referenced in the text
- [ ] Captions are self-explanatory (reader can understand without reading the body)
- [ ] Figures don't duplicate tables (choose one)
- [ ] Axes are labeled with units
- [ ] Statistical details are in tables, not prose (where practical)
- [ ] Resolution and format meet journal requirements

## Methods Rigor

- [ ] Study design is named explicitly (RCT, cohort, case-control, survey, etc.)
- [ ] Sample size and selection criteria are stated
- [ ] Variables, measures, and instruments are defined
- [ ] Statistical tests are named and justified
- [ ] Software and versions are specified
- [ ] Enough detail for reproduction by a competent researcher

## Reporting Guidelines

Apply the relevant guideline for your study type:

| Study Type | Guideline | URL |
|---|---|---|
| Randomized trial | CONSORT | consort-statement.org |
| Observational study | STROBE | strobe-statement.org |
| Systematic review | PRISMA | prisma-statement.org |
| Case report | CARE | care-statement.org |
| Diagnostic accuracy | STARD | stard-statement.org |
| Qualitative research | COREQ / SRQR | equator-network.org |
| Animal research | ARRIVE | arriveguidelines.org |
| Clinical trial protocol | SPIRIT | spirit-statement.org |
| Quality improvement | SQUIRE | squire-statement.org |

If your study type isn't listed, check the EQUATOR Network (equator-network.org) for the appropriate guideline.

## Language & Style

- [ ] Consistent tense (past for methods/results, present for established facts and discussion)
- [ ] Active voice preferred where natural ("We analyzed" not "Analysis was performed")
- [ ] Jargon is defined on first use or avoided
- [ ] Acronyms are spelled out on first use
- [ ] No unnecessary hedging ("It could potentially be suggested that" -> "We suggest")
- [ ] No unsupported assertions ("It is well known that" — cite it or cut it)
- [ ] Paragraphs have topic sentences and logical flow

## Ethics & Declarations

- [ ] Ethics/IRB approval stated (with approval number)
- [ ] Informed consent described
- [ ] Conflicts of interest declared (or "none" stated)
- [ ] Funding sources acknowledged
- [ ] Data availability statement included
- [ ] Author contributions statement (CRediT taxonomy if journal requires)
- [ ] Clinical trial registration number (if applicable)

## Formatting & Submission

- [ ] Title is specific, informative, and within journal word limit
- [ ] Abstract is self-contained (no citations, no abbreviations without definition)
- [ ] Abstract word count within journal limit (typically 150-300 words)
- [ ] Keywords provided (3-6, specific enough to aid discovery)
- [ ] Manuscript word count within journal limit
- [ ] Reference count within journal limit (if applicable)
- [ ] Cover letter written (why this journal, what the paper contributes)
- [ ] All supplementary materials are referenced in the main text
- [ ] Blinding requirements met (if journal requires anonymous submission)
