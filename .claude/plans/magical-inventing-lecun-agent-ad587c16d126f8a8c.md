## Execution Plan

### Phase 1: Parallel web searches (5 topic clusters, ~4 searches each = ~20 searches)
Run all 5 topic clusters in parallel using WebSearch. For each result, identify the 2–3 most promising URLs.

### Phase 2: Parallel deep fetches
For each topic, fetch the 2–3 best URLs found in Phase 1. Extract: names, dates, quotes, institutions, specific moments.

### Phase 3: Fill gaps
If any topic lacks a strong anecdote after Phase 2, run targeted follow-up searches (e.g., specific book titles, specific researcher names surfaced in Phase 2).

### Phase 4: Write output file
Synthesize all findings into structured brief at target path. Format:
- Topic header
- Anecdote(s) in narrative form (3–5 sentences each)
- Source citation (author, title, year, publication)
- Confidence level: HIGH / MEDIUM / LOW
- "Insider significance" and "general reader hook" notes
