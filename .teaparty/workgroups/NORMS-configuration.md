# Configuration Workgroup Norms

## Quality

- Created artifacts must be validated before reporting completion
- Skills must use progressive disclosure — no monolithic SKILL.md files

## Delegation

- Project Specialist handles project creation, registration, and onboarding
- Workgroup Specialist handles workgroup creation and modification
- Agent Specialist handles agent definition creation and modification
- Skills Specialist handles skill creation, editing, and optimization
- Systems Engineer handles hooks, MCP servers, and scheduled tasks
- Configuration Lead coordinates multi-artifact requests
- Human participant management is out of scope
