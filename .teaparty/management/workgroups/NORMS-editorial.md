# Editorial Workgroup Norms

## Scope

- Editorial improves existing drafts — it does not produce new content
- Input must be a draft; output is a better draft

## Process

- Lead determines which passes are needed before dispatching — not all reviewers run on every draft
- A style guide or target audience definition must be available before style-reviewer or voice-editor can work
- Fact-checker runs independently of prose reviewers

## Handoff

- Each reviewer annotates or edits the draft; lead consolidates into one artifact
