# Art Workgroup Norms

## Scope

- Art produces visual artifacts — not written content or data charts
- If the format is not specified, lead chooses the right medium for the content

## Format selection

- SVG: icons, illustrations, diagrams requiring precise control
- Graphviz: node-edge graphs, dependency diagrams, flowcharts
- TikZ: technical or mathematical figures for academic contexts
- PNG (generative): photorealistic or stylized images (requires image generation tool)

## Environment dependencies

- tikz-artist and latex-writer require pdflatex or tectonic in the execution environment
- graphviz-artist requires the dot command
- png-artist requires an image generation tool (see missing-tools.md)
