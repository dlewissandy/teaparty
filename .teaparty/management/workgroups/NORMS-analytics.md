# Analytics Workgroup Norms

## Scope

- Analytics analyzes data it receives — it does not collect or move data between systems
- Data engineering (moving and transforming data) is a separate concern

## Process

- Frame the analytical question precisely before dispatching — an ambiguous question produces an ambiguous answer
- data-scientist handles analysis; data-visualizer handles charts — they are not interchangeable

## Output

- Findings must include the analytical question, the method used, and a clear answer with evidence
