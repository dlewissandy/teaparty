# Quality Control Workgroup Norms

## Scope

- QC verifies completed work — dispatch after implementation, not during it
- Acceptance criteria must be defined before QC begins; lead requests them if missing

## Verdicts

- Lead reports a pass/fail verdict with evidence — not only when everything passes
- A "fail" with clear evidence is a complete deliverable

## Coverage

- Not all reviewers run on every artifact — lead selects appropriate checks for the work type
